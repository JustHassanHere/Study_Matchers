Study Matcher

A full-stack university training reservation platform built for students to discover and reserve field training opportunities online.

The system helps universities organize training programs and allows students to reserve available slots through a modern web platform.

Features
✔ Student authentication with JWT
✔ Role-based access control
✔ Admin dashboard
✔ Training field management
✔ Training reservation system
✔ MongoDB database integration
✔ FastAPI backend architecture
✔ React frontend
✔ Protected API routes
✔ Secure password hashing

Project Goal

The goal of this project is to simplify the process of reserving university field training opportunities by providing:

✔ Easy student registration
✔ Online reservation system
✔ Secure authentication
✔ Admin management tools
✔ Organized training listings
System Roles
Student
✔ Register account
✔ Login securely
✔ View training opportunities
✔ Reserve training slots
Admin
✔ Create training fields
✔ Manage training capacity
✔ Control reservations
✔ Manage platform data
